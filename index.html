Welcome to my personal CSS styles cheat sheet.

In this document I will list common css syntax and their uses.

syntax defined inputs
margin: Refers to the area surrounding an element's border. Positive integers measured in pixels(px). Example: margin: 20px; margin-(top, left, bottom, right):
margin: 0px 0px;
margin: 0px 0px 0px 0px;
margin: 0 auto;
padding: Refers to the space between an element and its border. padding-(top, left, bottom, right):
padding: 0px 0px;
padding: 0px 0px 0px 0px;
padding: 0 auto;
overflow: When content becomes too large for its container (usually due to page re-sizing) it determines how that content will be viewed. overflow: scroll; (make overflow content viewable via scroll bars
overflow: hidden; (overflow content will be hidden from view)
overflow: visible; (default value)
auto This key-word can be used to horizontally center and element. margin: 0 auto; (horizontally center element)
