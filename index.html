Welcome to my personal CSS styles cheat sheet.

In this table I will list common css syntax related to poistioning.

syntax defined
position: fixed; When a CSS position is set to fixed, it is set/pinned to a specific spot on a page.
position: relative; The value relative of the CSS position property enables an element to be positioned relative to where it would have originally been on a web page.
position: absolute; The value absolute enables an element to ignore sibling elements and instead be positioned relatives to its closest parent element this is positioned with relative or absolute.
overflow: This property can be set to display, hide, or scroll, and dictates how HTML will render content that overflows its parent's content area.
visibility The visibility property can hide or show elements.
